Weather Icons

This website will show you 10 different weather icons and give an explination of what they mean.

Sun

This icon is a classic rendition of a sun. This icon would be used on a beautiful sunny day!

Cloud

This is a cloud icon. It would be depicted on a day where the clouds completley cover the sky.

Moon

After the sun sets this moon icon would be illustrated to display when the moon is visible and out in the sky.

Partly Cloudy

This is a partly cloudy icon. Most days the sun peers out from behind the cloudys but still shines through, this icon depicts that.

Rain Cloud

A rain cloud icon would be displayed on a very rain day.

Rain Drop

A rain drop icon would be displayed when there was a rain shower as apossed to an extremly rainy day.

Lightning

A lightning icon would be shown during a thunderstorm.

Snow Cloud

A snow cloud icon is dipicted during a snowstorm.

Snowflake

A snowflake icon would be illustrated when there is light snow.

Wind

A wind icon would be displayed during a windstorm.
